#Análise de vendas com Python

Este projeto é uma análise de dados simples e também interativa  desenvolvida como parte dos meus estudos em Engenharia de software.

## Ver Online
O projeto está rodando ao vivo na nuvem. Clique abaixo para testar:
**[Acessar dashboard de vendas](https://pebkzpsvuvdp7eg.streamlite.app/)**

## Tecnologias usadas
* **Python 3.12**
* **Pandas** (Análise de dados)
* **Matplotlib** (Visualização de gráficos)
* **Ambiente Linux Mint**
* **Frontend:** Streamlit
* **Infraestrutura:** Streamlit Cloud


## O que o código faz 
1. Lê a planilha de vendas ("vendas.csv").
2. Processa os dados para separar dias e valores.
3. Gera um gráfico de barras visualizando o desempenho semanal.

## Desafios superados
Este projeto foi configurado em um hardware legado (Atom D525), exigindo gerenciamento avançado de dependencias e ambientes virtuais para rodar bibliotecas modernas de Data science.
